Welcome to Beyond the Arc

This is a modest little site built with Django on top of an existing database from a Rails app. It displays data for upcoming NBA games. The data can be assumed to be totally incorrect as it hasn't been updated completely. However this is just an example of what 8 hours with Django can result in.

The Stats

You can see the NBA games scheduled for today. Listed for each game is each team's Points per game (PPG), PPG home and away, a defensive modifier which is a cumulative statistic based on the team's prior performance and the predicted points for the team based on their PPG and the opponent's defensive modifier.
